Numeric Root Finding Methods

There are two types:

  1. Bracketing methods (bisection method) 二分法

    二分的需要一个初始区间

  2. Open methods (Newton-Raphson method) 牛顿法

    牛顿法就只需要一个初始的点去迭代

Root are found iteratively until some criteria are satisfied:

  • Accuracy 精准度足够
  • Number of iteration 迭代做完

二分法

Assumptions:

  • f(x)f(x)f(x) continuous on l,u{l,u}l,u
  • f(l)⋅f(u)<0f(l) · f(u) < 0f(l)⋅f(u)<0

二分本质：找边界点

给定某个区间在区间上定义了某种性质，使得此性质在右半边区间是满足的，在左半边区间是不满足的，那么二分就可以寻找边界，既可以找不满足的边界也可以找满足的边界。

牛顿法

Assumption:

  • f(x)f(x)f(x) continuous
  • f′(x)f'(x)f′(x) known

General Formula：

Xn=Xn−1−f(xn−1)f′(xn−1)X_n = X_{n-1} - \frac{f(x_{n-1})}{f'(x_{n-1})} Xn​=Xn−1​−f′(xn−1​)f(xn−1​)​

牛顿迭代法又称为切线法，它比一般的迭代法具有更高的收敛速度。

首先，选择一个接近函数f(x)f(x)f(x)零点的x0x_0x0​，计算相应的f(x0)f(x_0)f(x0​)和切线的斜率f’(x0)f’(x_0)f’(x0​)。然后我们可以表示出来，过(x0,f(x0))(x_0,f(x_0))(x0​,f(x0​))的切线方程为：

y=f(x0)+f′(x0)(x−x0)y =f(x_0) + f'(x_0)(x - x_0) y=f(x0​)+f′(x0​)(x−x0​)

和x轴交点的x的坐标，也就是如下方程的解：

f(x0)+f′(x0)(x−x0)=0f(x_0) + f'(x_0)(x - x_0) = 0 f(x0​)+f′(x0​)(x−x0​)=0

Matlab中的实现求解

在matlab中有一个fsolve() function 去实现对非线性方程组求解。

For example, solve this equation:

f2=1.2x+0.3+xsin(x)f2 = 1.2x + 0.3 + xsin(x) f2=1.2x+0.3+xsin(x)

当然，这里的fsolve并不是使用的纯粹的牛顿迭代，比普通牛顿迭代更加强大。具体实现不再深究。

两者比较

牛顿法比二分的速度来讲要快，但是如果function有问题的话，不会收敛，会diverge（发散）。而且牛顿法需要微分。
